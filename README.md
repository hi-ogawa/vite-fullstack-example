# vite-fullstack-example (wip)

## tools

- rendering
  - react
  - vite-plugin-ssr
- api
  - trpc
  - react-query
- database
  - query builder: drizzle
  - schema migration: prisma
- testing
  - unit: vitest
  - e2e: playwright
- deployment
  - vercel cdn/serverless

## usage

```sh
pnpm i
make db/reset
pnpm dev
pnpm test
pnpm test-e2e
```

# vite-fullstack-example (wip)

## tools

- rendering
  - react
  - vite-plugin-ssr
  - unocss
- api
  - trpc
  - tanstack query
- deployment
  - vercel serverless, cdn

## usage

```sh
# development
pnpm i
pnpm dev

# release (see also misc/vercel/README.md)
pnpm build
pnpm release
pnpm release-production
```

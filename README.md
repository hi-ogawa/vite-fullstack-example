# vite-fullstack-example (wip)

## tools

- rendering
  - react
  - vite-plugin-ssr
- api
  - trpc
  - react-query
- database
  - postgres
  - drizzle (query builder)
  - prisma (schema migration)
- testing
  - vitest (unit)
  - playwright (e2e)
- deployment
  - vercel cdn/serverless

## usage

```sh
# development
pnpm i
pnpm dev

# release
pnpm build
pnpm release
pnpm release-production
```

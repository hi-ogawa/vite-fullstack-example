# vite-fullstack-example

## tools

- rendering
  - react
  - vite-plugin-ssr
  - unocss (css)
- api
  - trpc
  - react-query
- TODO: database
  - postgres
  - drizzle (query builder)
  - prisma (schema migration)
- TODO: testing
  - vitest (unit)
  - playwright (e2e)
- deployment
  - vercel cdn/serverless

## usage

```sh
# development
pnpm i
pnpm dev

# release
pnpm build
pnpm release
pnpm release-production
```
